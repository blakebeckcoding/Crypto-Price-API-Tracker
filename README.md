Crypto-Price-API-Tracker

This is just a learning project, I used a youtube video which I'll link in the README.

It's a simple crypto price tracker that can connect to any API and fetch data, it refreshes every 30 secs but it can be changed in the code.

Youtube video I used to do this code (I did some tweaks to fix and customize it): https://youtu.be/AB4pBWgmToc?si=Cawh5yUx12oVL1Qt

Original code credits go to Coding with Evan

Features:

Can track any crypto coins
Can connect to any API (meaning any coin or any API you want really)
You can set the code to refresh whenever you want

Requirements:

Python 3.10+
requests library
tkinter (included in standard Python)

Optional (if you want to build a .EXE file from this):

Pyinstaller
